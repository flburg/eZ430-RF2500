Device usually connects to Lenovo on port 6.

Start Realterm
	Go to port tab, and select 57600, no parity, 8 data, 1 stop, no flow control
                Click the Open button – should see a stream of samples printed on the  console
                                If the remote is on, should see samples from HUB0 and 0002
                Go to the Capture tab
                                Set the File to point to capture.txt where the Tcl script is
                                Click Start Overwrite

In a cmd window in Desktop
                cd to where the tcl script lives
                type ‘tclsh <name of script>’ (also can use wish)
